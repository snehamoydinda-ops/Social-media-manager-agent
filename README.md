# Social Media Manager Agent

This project includes an AI powered Social Media Manager Agent that aids users
to develop different types of content for use on various social media platforms.

## Features
- Creates captions for posts
- Suggests "hashtags
- Rewrites the content for various platforms
- Suggests posting time and schedule

## How it works

The user provides a topic or description.
The agent produces optimized social media content.

## Tech Stack

- Python
- OpenAI / LLM API
- Command Line Interface

## How to run

1. Clone the repository git clone https://github.com

2. Installing the Requirements pip install -r requirements.txt

3. Run python main.py
   
## Example
Input:
Promote a college tech fest

Output:
Instagram caption, hashtags and posting suggestion.
