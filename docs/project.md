# Project Documentation

## Project Name
Social Media Manager Agent

## Overview
The Social Media Manager Agent helps users generate social media captions
and hashtags for different platforms such as Instagram, LinkedIn and Twitter.

## Objective
The main objective of this project is to reduce the time required to write
platform specific social media content.

## System Workflow

User provides topic
→ Agent processes the input
→ Platform specific caption is generated
→ Hashtags are generated
→ Final output is displayed to the user

## Modules

1. Input Module  
Takes topic and platform name from the user.

2. Content Generator Module  
Generates captions and hashtags for the selected platform.

3. Output Module  
Displays the generated content to the user.

## Technology Used

- Python
- Command line interface
